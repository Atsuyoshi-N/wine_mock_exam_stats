# ワイン受験.com 模擬試験 成績ダッシュボード

[ワイン受験.com](https://www.wine-jyuken.com/) の模擬試験結果を記録・分析するための Web ダッシュボードです。  
Chrome拡張「ワイン受験.com Anki変換」と連携して使用します。

## 機能

- 試験ごとの得点・正答率・偏差値・判定・正解問題数・不正解問題数の確認
- スコア推移グラフ（全試験）
- 分野別成績の横棒グラフ＋テーブル（正解数・不正解数・得点率でソート可）
- 分野別推移グラフ
- スマートフォン・タブレット対応

## 使い方

### 初回セットアップ

1. Chrome拡張「ワイン受験.com Anki変換」をインストール
2. 拡張機能のオプションページを開く（`chrome://extensions/` → 詳細 → 拡張機能のオプション）
3. GitHub Personal Access Token（`gist` スコープのみ）を入力して「保存」
4. 「今すぐ同期」をクリック → Gist ID が発行される
5. このダッシュボードを開き、Gist ID を入力して「読み込む」

### 2回目以降の同期

採点ページを開くと**自動的に**最新データが GitHub Gist に同期されます。  
ダッシュボードの「**更新**」ボタンをタップすれば最新データを反映できます。

手動で同期したい場合（例：複数のデバイスから操作した場合など）：

1. Chrome 拡張のオプションページを開く
2. 「今すぐ同期」ボタンをクリック
3. ダッシュボードの「更新」ボタンをクリック

### Gist ID を変更する

別のアカウントのデータを見たい場合など：

1. ダッシュボード右上の「別のGistを使う」ボタンをクリック
2. 新しい Gist ID を入力
